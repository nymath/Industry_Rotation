| 参数 | 含义 |
| --- | --- |
|start = '2016-01-01' |
|end = '2021-10-01' | 建议不要超过这个时间，因为我的国债数据只有到去年12月的

这个是我们用哪个预测值，可以用ctrl+F搜一下参数位置，
可选参数有 "VAR", "fbayes", "ns", "ridge", "真实值", "平均值"
